# 日内交易剧本 v2 — 模板规范

---

## 设计原则

> **结论先行，数据后附。**
> 交易员打开 playbook 的第一秒就应该知道：做什么方向、在哪里入场、止损多少、目标多少。
> 所有分析逻辑和数据雷达放在底部，供需要深入理解的人参考。

---

## 模板结构

```
━━━ {TICKER} | 日内剧本 ━━━
⏰ {TIMESTAMP} ET

┌─────────────────────────────────┐
│  🎯 {方向} {入场价} → TP {目标}  │
│     止损 {止损价} | R:R {X:X}    │
└─────────────────────────────────┘

▸ 置信度: {做多 X% / 做空 Y%}
▸ 日型: {趋势日 / 震荡日 / V型反转 / 缺口追击 / 缺口回补 / 窄幅磨损}
▸ 模式识别: {当前价格结构的一句话描述}
▸ vs 上一版: {具体变化, 如 "方向从观望→偏多" / "入场位下移 0.3%"}

── 主方案 ──────────────────────
📈/📉 A: {方案名}
  方向: {做多/做空}
  触发: {具体条件, 一句话}
  入场: {价格} ({距当前价 X%})
  止损: {价格} ({理由}) | {占 5min ATR 的倍数}
  TP1:  {价格} (50%) | {距入场 X%} | {历史达成率或ATR校验}
  TP2:  {价格} (清仓) | {距入场 X%}
  R:R:  {X:X} (有效R:R, 已校验止损噪音和目标可达性)
  合约: {期权建议, 如适用}

── 对冲方案 ────────────────────
📉/📈 B: {方案名}
  方向: {与主方案相反}
  触发: {主方案失效的具体条件}
  入场: {价格} ({距当前价 X%})
  止损: {价格}
  TP1:  {价格}
  R:R:  {X:X}

── 近端备选 ────────────────────
⚡ C: {当前价位直接可做的方案}
  说明: {当 A/B 入场位较远时, 提供一个当前价附近的轻仓试探方案}
  入场: {当前价附近}
  止损: {紧止损}
  TP1:  {保守目标}
  仓位: {主方案的 30-50%}

── 失效与切换 ──────────────────
🚫 失效条件:
  • 主方案失效: {具体价位 + 量能条件}
  • 日型切换: {从当前日型切换到另一日型的条件}
  • 强制止损: {极端情况下的硬止损}

─────────────────────────────────
📊 数据参考 (折叠区)
  价格: {当前价} | VWAP {价格} ({偏离%})
  量能: RVOL {X} ({描述}) | {开盘效应校正后的值}
  波动: 振幅 {X%} | 预估剩余波动 {X%} | 5min ATR {X}
  结构: VAH {X} | POC {X} | VAL {X} ({周期})
       PDH {X} | PDL {X} | PMH {X} | PML {X}
  期权: Call Wall {X} | Put Wall {X} | MaxPain {X} | IV比 {X}
  相对强度: vs SPY {强于/弱于/同步} ({过去N根K线相关性})
  自适应: P16d 阈值 {X} (rank {X%})
━━━━━━━━━━━━━━━━━━━━
```

## 各字段规范与约束

### 1. 核心结论区（顶部方框）

- **必须在第一行给出方向 + 入场价 + 目标**，不允许"观望"作为核心结论持续超过 60 分钟
- 如果确实无法判断方向，格式为：

```
┌──────────────────────────────────────┐
│  ⏳ 观望 → 60min 后强制归类           │
│  偏多触发: {条件}  偏空触发: {条件}    │
└──────────────────────────────────────┘
```

- 观望状态下一次 playbook 更新时（通常 60-90 分钟后），**必须**给出方向性结论
- "观望"连续出现 2 次 → 系统报错，强制基于现有数据归类


### 2. 置信度

- 格式: `做多 65% / 做空 35%` 或 `做多 80% / 做空 20%`
- **禁止** `50% / 50%` 持续超过 1 个版本
- 置信度来源需标注（如 "VWAP上方+RVOL趋势级 → 偏多"）


### 3. 日型判断

标准日型分类（6 种，互斥）:

| 日型 | 特征 | 核心策略 |
|------|------|----------|
| 📈 趋势日（强） | 单边运行, RVOL>1.5, VWAP单侧 | 回调顺势 |
| 📈 弱趋势日 | VWAP缓慢单向移动, 价格持续单侧 | VWAP附近顺势轻仓 |
| 📦 震荡日 | 价格围绕VWAP反复穿越, RVOL<1 | 边缘均值回归 |
| 🔄 V型反转 | 开盘单边→急速反转→反方向运行 | 反转确认后顺新方向 |
| 🚀 缺口追击 | 跳空+方向延续, 不回补 | 回调接顺势仓 |
| 🔻 缺口回补 | 跳空后反向回补, 量能衰减 | 回补方向做 |
| 😴 窄幅磨损 | ATR极低, 无方向, RVOL<0.5 | 不做或极轻仓 |

**日型切换规则:**
- 开盘 30 分钟内允许 "待定"
- 30 分钟后必须初步归类
- 价格结构出现明确变化时（如V型反转）, 必须在下一个 playbook 中更新日型
- 日型切换时必须标注: `⚠️ 日型切换: {旧} → {新} | 原因: {一句话}`


### 4. 入场可执行性约束

**硬性规则:**

```
if 入场位距当前价 > 预估剩余波动 × 50%:
    该方案标记为 "远端方案"
    必须同时生成一个 "近端备选 (Plan C)"

if 入场位距当前价 > 预估剩余波动 × 100%:
    该方案标记为 "⛔ 当日大概率不可达"
    降级为参考, 不作为首选
```

- 首选方案的入场位应在 30 分钟内有合理触发概率
- 每份 playbook 必须包含至少一个 "距当前价 < 0.3%" 的可执行方案


### 5. 止损规范

**最小止损 = max(1.5 × 5min ATR, 标的日均波动 × 5%)**

示例:
- TSLA 5min ATR = $1.20 → 最小止损 $1.80
- SPY 5min ATR = $0.30 → 最小止损 $0.45

止损设置需标注:
```
止损: 396.72 (PMH) | 1.8x 5min ATR ✓
止损: 396.72 (PMH) | 0.3x 5min ATR ⚠️ 过紧, 噪音易触发
```


### 6. 目标 & R:R 规范

**目标校验:**
```
if TP1距入场 > 预估剩余波动:
    标注 "⚠️ 目标可能超出当日剩余空间"
    建议缩小目标至: 入场价 ± 预估剩余波动 × 70%

if TP1距入场 > 预估剩余波动 × 150%:
    标注 "⛔ 目标不现实, 已自动调整"
    强制调整 TP1
```

**有效 R:R 计算:**
```
有效R:R = (TP1达成概率 × TP1收益) / (止损触发概率 × 止损亏损)

# 简化版: 如果止损 < 1x 5min ATR, 止损触发概率上调至 60%+
# 如果 TP1 > 剩余波动, TP1 达成概率下调至 30%-
```

- R:R < 1.5:1 的方案不允许作为首选
- R:R > 8:1 需标注 "⚠️ 极端R:R, 请校验止损是否过紧或目标是否过远"


### 7. 自适应 & 版本对比

每份 playbook 必须包含 `vs 上一版` 字段:

```
▸ vs 上一版: 方向从观望→偏多 | 日型从不明确→弱趋势日 | 入场位从 675.00→672.50
```

如果与上一版无实质变化, 则标注:
```
▸ vs 上一版: ⚠️ 无实质变化 — 考虑是否需要本次更新
```


### 8. 相对强度（个股专用）

```
▸ 相对强度: vs SPY 显著走强 (+1.2% vs -0.3%) | 30min 相关性 0.12 (脱钩)
  → 大盘方向权重下调, 个股独立逻辑优先
```

**规则:**
- 当个股与 SPY 30 分钟滚动相关性 < 0.3 时, 标注 "脱钩"
- 脱钩状态下, "大盘偏X, 个股逆势" 的警告自动降级为灰色提示
- 当个股逆大盘走强/走弱超过 0.5% 时, 视为强信号, 应纳入方向判断


### 9. RVOL 开盘校正

```
if 时间 < 09:45 ET:
    RVOL显示为: "RVOL {原始值} (⚠️ 开盘放大, 校正值约 {X})"
    校正方法: 用同时段历史RVOL中位数做比值

if 时间 >= 10:00 ET:
    RVOL正常显示
```

## 完整示例: 理想的 TSLA Playbook

```
━━━ TSLA | 日内剧本 ━━━
⏰ 2026-03-17 10:05:12 ET

┌───────────────────────────────────────┐
│  📈 做多 394.50 → TP 396.72 (PMH)    │
│     止损 392.80 | R:R 1:1.3 (有效)    │
└───────────────────────────────────────┘

▸ 置信度: 做多 65% / 做空 35%
  (V型反转确认: 开盘下探393后快速收回VWAP, RVOL维持>1.5)
▸ 日型: 🔄 V型反转
  ⚠️ 日型切换: 待定 → V型反转 | 原因: 开盘30min下探后强势收回VWAP上方
▸ 模式识别: 开盘杀跌至393触底, 10:00前收回VWAP并站稳, 经典V反结构
▸ vs 上一版: 方向从观望→偏多 | 日型从待定→V型反转 | 新增做多入场

── 主方案 ──────────────────────
📈 A: V型反转做多
  方向: 做多
  触发: 价格站稳 VWAP 394.50 上方 + 回踩不破
  入场: 394.50 (距当前价 0.1%)
  止损: 392.80 (日低393下方) | 1.4x 5min ATR ✓
  TP1:  396.72 (PMH) (50%) | 距入场 0.56% | 在预估剩余波动内 ✓
  TP2:  400.00 (整数关口) (清仓) | 距入场 1.4%
  R:R:  1:1.3 (TP1) / 1:3.2 (TP2)

── 对冲方案 ────────────────────
📉 B: 反转失败做空
  方向: 做空
  触发: 价格再次跌破 VWAP + 回测 VWAP 失败
  入场: 394.20 (VWAP 下方确认)
  止损: 395.50 (VWAP + 1x ATR 上方)
  TP1:  393.00 (日低回测)
  R:R:  1:0.9 → 仅轻仓试探 (30% 仓位)

── 近端备选 ────────────────────
⚡ C: 当前价位轻仓做多
  说明: 若价格已在 394.80+ 且不愿等回踩
  入场: 394.80 (当前价附近)
  止损: 393.80 (VWAP 下方)
  TP1:  396.72 (PMH)
  仓位: 主方案的 40%

── 失效与切换 ──────────────────
🚫 失效条件:
  • 主方案失效: 跌破 393.00 (日低) + 放量 → V反失败
  • 日型切换: 若 11:00 前无法突破 PMH 396.72, 降级为震荡日
  • 下行风险: 跌破 PML 393.42 → 切换为下行趋势日

─────────────────────────────────
📊 数据参考
  价格: 394.60 | VWAP 394.50 (+0.03%)
  量能: RVOL 1.82 (偏强, 开盘校正后)
  波动: 振幅 0.91% | 预估剩余波动 1.8% | 5min ATR $1.20
  结构: VAH 405.00 | POC 395.50 | VAL 390.00 (5d)
       PDH 403.73 | PDL 394.42 | PMH 396.72 | PML 393.42
  相对强度: vs SPY 走强 (+0.3% vs -0.1%) | 30min相关性 0.35 (偏弱)
    → 大盘弱势对本剧本影响有限
  自适应: P16d 阈值 1.14 (rank 85%)
━━━━━━━━━━━━━━━━━━━━
```

## 系统级约束 (Checklist)

每份 playbook 生成前, 系统必须通过以下检查:

- [ ] 核心结论是否包含方向 + 价位? (观望不超过60min)
- [ ] 首选方案入场位距当前价 < 预估剩余波动 × 50%?
- [ ] 是否包含反向对冲方案 (带具体价位)?
- [ ] 止损 > 1.5x 5min ATR?
- [ ] TP1 < 预估剩余波动?
- [ ] R:R ≥ 1.5:1 (首选方案)?
- [ ] vs 上一版: 是否有实质变化?
- [ ] 开盘30min内 RVOL 是否已校正?
- [ ] 个股是否已计算相对强度?
- [ ] 日型是否超过60min未归类?
